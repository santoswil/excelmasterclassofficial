Esse curso é para você que deseja aprender:

Funções e Utilidades


Nesta aula você aprenderá a fazer os primeiros ajustes na tela inicial do Excel, irá conhecer as funções mais importantes e simples de serem utilizadas e alguns modelos de planilhas e gráficos que você conseguirá criar.

Criar Planilhas Profissionais


Nesta aula será apresentado todo o passo a passo para você passar os seus dados corretamente para o Excel e transformar em uma planilha bem elaborada, com um belo design.

Criar Gráficos Profissionais


Nesta aula você irá aprender todo o passo a passo para você criar Gráficos a partir das sua base de dados (planilhas/tabelas). Um bom gráfico precisa ter os dados organizados e visíveis.
